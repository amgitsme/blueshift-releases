# Blueshift SEA — Releases

Downloads are on the **[Releases page](https://github.com/amgitsme/blueshift-releases/releases/latest)**.

- **Installer** — `Blueshift-SEA-Setup-<version>.exe` (recommended). Installs the app and then
  updates itself automatically; you won't need to download again.
- **Portable** — `Blueshift-SEA-<version>.exe`. Single file, no install. Does **not** auto-update —
  re-download from Releases for new versions, or switch to the installer to get automatic updates.

Binaries are attached to each Release, not committed to this repository.
